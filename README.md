## motivação

Este projeto surgiu a partir da minha experiencia prateica como ajudante de eletricista, onde frequentimente era necessario dimensionar disjuntores e 
condutores com base na carga total do circuito.

A ideia foi transferir esse raciocionio, automatizando ele em uma aplicação simples, evitando erros manuais comuns.

##Contexto deste projeto:

Este projeto foi criado como um exercício de lógica e programação, aplicando conceitos reais 
de eletricidade em uma solução web simples.
